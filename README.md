# Practo Clone – Online Doctor Consultation UI

A fully designed frontend clone of Practo’s online doctor consultation platform.<br>
This project focuses on building a real-world healthcare UI with multiple sections, structured layout, and user-focused design.

<br>

## 📌 Project Overview

This project replicates a doctor consultation platform interface where users can:<br>

Browse doctors<br>
Consult online<br>
Explore specialities<br>
View health concerns<br>
Read FAQs and testimonials

<br>

It is built to practice frontend development, UI structuring, and responsive design.

<br>

## 🚀 Features

🧭 Navigation bar with dropdowns<br>
🏥 Hero section with CTA (Consult Now)<br>
👩‍⚕️ Specialities section (25+ categories UI)<br>
❤️ Common health concerns cards<br>
🎁 Offers section<br>
👨‍⚕️ Doctors listing carousel<br>
⚙️ “How it works” step section<br>
📊 Stats and benefits section<br>
🎥 Video consultation section (UI)<br>
💬 Testimonials carousel<br>
❓ FAQs section<br>
📱 App download section<br>
📚 SEO content section<br>
🧾 Detailed footer

<br>

## 🛠️ Technologies Used

HTML5<br>
CSS3
