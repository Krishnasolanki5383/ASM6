Practo Clone – Online Doctor Consultation UI

A fully designed frontend clone of Practo’s online doctor consultation platform. This project focuses on building a real-world healthcare UI with multiple sections, structured layout, and user-focused design.

📌 Project Overview

This project replicates a doctor consultation platform interface where users can:

Browse doctors
Consult online
Explore specialities
View health concerns
Read FAQs and testimonials

It is built to practice frontend development, UI structuring, and responsive design.
